        # 碎碎念

        本部分是作者的碎碎念

        为什么项目名字叫做碎遮

        取自《有匪》一书女主用过的第二把刀的名字 中二 (确实想不出啥好名字了 天幕如遮，唯我一刀可碎千里华盖，纵横四海而无阻
        是谓碎遮

        为什么没有使用前后端分离

        因为作者准备考研，时间比较少，对于前后端分离（主要是前端）还不熟练，所以没有使用
